# Smelly Tofu

[![Build and Deploy](https://github.com/cnphil/blog/actions/workflows/deploy.yml/badge.svg)](https://github.com/cnphil/blog/actions/workflows/deploy.yml)

Source code for [blog.phil.tw](https://blog.phil.tw/), statically generated by [Hakyll](https://jaspervdj.be/hakyll/).

## Usage

```
stack build
stack exec site build
```

The generated site will be in `_site/`.

## Deployment

Pushes to `master` automatically build and deploy to blog.phil.tw via GitHub Actions.
